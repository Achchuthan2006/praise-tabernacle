# Digital magazine (PDFs)

Create your monthly magazine in Canva (or similar), export as PDF, then place the PDF file(s) in this folder.

Recommended naming:

- `YYYY-MM.pdf` (bilingual)
- `YYYY-MM-en.pdf` and `YYYY-MM-ta.pdf` (if split by language)

Optional cover images:

- `YYYY-MM-cover.webp` (or `.jpg/.png`)

After adding files, update the issue list in `lib/magazine.ts`.
